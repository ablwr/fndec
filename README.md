# fndec

Created for Artefactual Systems Inc.

Guess filename encodings... based on @richardlehane's [characterize][char]
golang package.

## Usage
```
    ross-spencer@artefactual:~:$ fndec -h
    Usage of fndec:
      -file string
    	    File to read the filename from.
      -version
    	    Return version information.

```
